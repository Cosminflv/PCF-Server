Authentication

Register a new user

Register a user by providing a unique username and password. Returns the created user details upon success.

Request Body schema: application/json
required
username
required
string (Username)
password
required
string (Password)

Responses

http://127.0.0.1:8000/register

Request samples

  • Payload
Content type
application/json
{
  • "username": "john_doe",
  • "password": "secretpassword123"
}

Response samples

  • 201
  • 422
Content type
application/json
{
  • "id": 1,
  • "username": "john_doe"
}

Upload a photo

Upload a photo file with a gallery password and optional subject name. If subject_name is 'noSubject', the system will try to predict it automatically.

Authorizations:
HTTPBearer
Request Body schema: multipart/form-data
required
file
required
string <binary> (File)
gallery_password
required
string (Gallery Password)
Subject Name (string) or Subject Name (null) (Subject Name)

Responses

http://127.0.0.1:8000/photos/

  • 422

Apply a filter to a photo

Apply an image filter to a photo. Supported filters: none, sepia, black and white, color inversion. If 'none' is provided, the original image is restored. Requires gallery password.

Authorizations:
HTTPBearer
path Parameters
photo_id
required
integer (Photo Id)
Request Body schema: application/x-www-form-urlencoded
required
filter_name
required
string (Filter Name)
gallery_password
required
string (Gallery Password)

Responses

http://127.0.0.1:8000/photos/{photo_id}/filter
